# Script — DAM, add Anthropic provider

DAM runs your AI coding agent in its own Kubernetes pod — credentials
safe, network sandboxed, awake when you're not.

Sign in {{username}} {{kc-login}}, and the get-started bar shows what's
next. Click "Set up a provider" {{set-up-a-provider}} — you land on
Anthropic.

You've got two ways to add your credential. OAuth Token plugs into the
session you get from running `claude setup-token` on your own machine.
Or paste an API key {{api-key}} directly, if you already have one
provisioned.

Pick the tab that matches what you have, drop it in — and you're ready
for connections and agents.
